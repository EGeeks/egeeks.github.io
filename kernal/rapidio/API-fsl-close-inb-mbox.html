Name

fsl_close_inb_mbox — Shut down MPC85xx inbound mailbox

Synopsis

void fsl_close_inb_mbox (struct rio_mport * mport,
 int mbox);
 

Arguments

mport

Master port implementing the inbound message unit

mbox

Mailbox to close

Description

Disables the inbound message unit, free all buffers, and frees the inbound message interrupt.
