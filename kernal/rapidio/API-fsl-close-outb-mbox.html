Name

fsl_close_outb_mbox — Shut down MPC85xx outbound mailbox

Synopsis

void fsl_close_outb_mbox (struct rio_mport * mport,
 int mbox);
 

Arguments

mport

Master port implementing the outbound message unit

mbox

Mailbox to close

Description

Disables the outbound message unit, free all buffers, and frees the outbound message interrupt.
