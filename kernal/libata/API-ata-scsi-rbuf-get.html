ata_scsi_rbuf_get
Prev Chapter 6. libata SCSI translation/emulation Next

Name

ata_scsi_rbuf_get — Map response buffer.

Synopsis

void * ata_scsi_rbuf_get (struct scsi_cmnd * cmd,
 bool copy_in,
 unsigned long * flags);
 

Arguments

cmd

SCSI command containing buffer to be mapped.

copy_in

copy in from user buffer

flags

unsigned long variable to store irq enable status

Description

Prepare buffer for simulated SCSI commands.

LOCKING

spin_lock_irqsave(ata_scsi_rbuf_lock) on success

RETURNS

Pointer to response buffer.
