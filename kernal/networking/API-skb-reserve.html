Name

skb_reserve — adjust headroom

Synopsis

void skb_reserve (struct sk_buff * skb,
 int len);
 

Arguments

skb

buffer to alter

len

bytes to move

Description

Increase the headroom of an empty sk_buff by reducing the tail room. This is only allowed for an empty buffer.
