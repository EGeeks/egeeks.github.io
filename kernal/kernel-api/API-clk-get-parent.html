Name

clk_get_parent — get the parent clock source for this clock

Synopsis

struct clk * clk_get_parent (struct clk * clk);
 

Arguments

clk

clock source

Description

Returns struct clk corresponding to parent clock source, or valid IS_ERR condition containing errno.
