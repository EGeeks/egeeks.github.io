Name

kfifo_initialized — Check if the fifo is initialized

Synopsis

kfifo_initialized ( fifo);
 

Arguments

fifo

address of the fifo to check

Description

Return true if fifo is initialized, otherwise false. Assumes the fifo was 0 before.
