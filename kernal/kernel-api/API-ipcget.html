Name

ipcget — Common sys_*get code

Synopsis

int ipcget (struct ipc_namespace * ns,

 struct ipc_ops * ops,
 struct ipc_params * params);
 

Arguments

ns

namsepace

ids

IPC identifier set

ops

Description

Common routine called by sys_msgget, sys_semget and sys_shmget.
